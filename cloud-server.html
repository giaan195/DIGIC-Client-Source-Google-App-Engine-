Giới thiệu dịch vụ Cloud Server

Dịch vụ Cloud Server là sự kết hợp của nhiều công nghệ tiên tiến, hiện đại nhất hiện nay.

  • Công nghệ ảo hóa Hyper-V tiên tiến, linh hoạt.
  • 100% ổ cứng SSD chuyên dụng.
  • Hạ tầng mạng Microsoft với kết nối 10Gbps
  • Hệ thống máy chủ chuyên dụng, Datacenter Microsoft

Cloud Server có tính năng SSD, CPU nhanh và cấu hình CPU-to-RAM tối ưu, dựa trên bộ xử lý 2.3 GHz Intel Xeon® E5-2673 v4 (Broadwell) và với công nghệ Intel Turbo Boost 2.0 có thể chạy với tốc độ lên đến 3,3 GHz.
